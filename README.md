# Unity Tools

Small quality-of-life editor tools for Unity.

## Installation

Add via **Package Manager → Install package from git URL**:

```
https://github.com/Maksatus/unity-tools.git
```

## Tools

| Tool | Shortcut | What it does |
|------|----------|--------------|
| [Scene Selector](Documentation~/SceneSelector.md) | ``Ctrl+` `` | Quick scene switcher with search, grouped by folder. Open scenes single or additively right from the list. |
| [Material Property Cleaner](Documentation~/MaterialCleaner.md) | — | Finds and removes stale properties that materials keep after shader changes. Can transfer Float values to Int properties. |
| [Rename Objects](Documentation~/RenameObjects.md) | ``Shift+` `` | Batch-renames selected objects and assets with live preview: prefix/suffix, replace, insert, lowercase, trim. |

All tools live under **Tools** in the main menu.
